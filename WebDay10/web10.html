DHTML

  • 简介： Dynamic(动态) HTML ， 这并不是新的技术，只是使用 HTML+CSS+JS得到一个动态页面效果称为DHTML
  • DHTML： 包括 DOM BOM

  • BOM：Browser（浏览器）Object（对象）Model（模型），在BOM中存在一些和浏览器相关的对象包括：location，document，history，screen，navigator

  • DOM：Document（文档）Object（对象）Model（模型），存放和页面相关的对象

window下的全局属性

  • location:位置 location.href 可以获取和设置当前浏览的页面地址 location.reload();//页面重新加载（刷新）

  • history:浏览历史

    history.length 得到当前窗口访问过的页面数量 history.back() 返回上一个页面 history.forward() 前往下一个页面 history.go(num) 前往某个页面 0代表当前页面 1代表下一个页面 -1代表上一个页面 2代表下两个页面

  • screen:屏幕 width/height: 得到当前屏幕的分辨率 availWidth/availHeight: 可用屏幕的宽高

  • navigator：导航/帮助 navigator.userAgent 得到客户端浏览器版本信息

  • document:文档（页面）

获取元素的几种方式

    /* 获取元素的几种方式 */
    /* 1. 通过id获取元素*/
    var input = 
        document.getElementById("myinput");
    /* 2. 通过标签名获取元素  得到的是数组 */
    var inputs = 
        document.getElementsByTagName("input");
    /* 3. 通过层级关系查找元素 */
    /* 3.1 得到上级元素(父元素) */ 
    var mydiv = input.parentElement;
    /* 3.2 得到子元素们 和 元素内部的文本内容*/
    var nodes = mydiv.childNodes;
    /* 得到子元素的数量  */
    var elementsCount = mydiv.childElementCount;
    /* 3.3 得到元素的兄弟元素们 */ 
    var nodes2 = input.parentElement.childNodes;
    /* 4. 通过name属性的值获取元素 */
    var radius = document.getElementsByName("gender");
    /* 5. 通过class属性的值获取元素 */
    var elements = document.getElementsByClassName("c1");
